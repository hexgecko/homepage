What is JavaScript?

It's the programming language of the web! It create interactivity like click, drag, tab, swipe, etc. Most commonly used for front-end development, but it's also used for end-end development as well, as Node.js. Don't use JavaScript for things like hover and focus event, because CSS can handle these thinks better.

DOM: Document Object Model, it's the outline of the page. There is a lot of methods for manipulating object on the page, the most common is:

  • document.getElementById: it's the fastes but there are others.
  • Lot of properties to change like: value, checked, className and id.
  • document.getElementById("id-of-element").value

To add a sumit button:

document.getElementById("sumbit-button-id").addEventListener("sumit"), function(evt) {
  evt.preventDefault(); // prevent a reload of the page
}

To set a default value if a null is returned, a or (||) can be used after the state, it's called sanitation, ex:

var value = parseInt(document.getElementById("element-id").value, 10) || 0;

To watch a contest of a tag with a given id the innerHTML member can be used, ex:

console.log(document.getElementById("element-id").innerHTML);

To set the attribute of an element the methods getAttribute(name) and setAttribute(name, value) can be used.

To create a new element the method document.createElement(name), then the new element can be added by the method _appendChild(element);. To update the text in the element a document.createTextNode(str); can be used an then the text is inserted with appendChilde(node);, below is an example:

var parentElement = document.getElementById('parent-element-id');
var newElement = document.createElement('li');
var textNode = document.createTextNode("List text");
newElement.appendChild(textNode);
parentElement.appendChild(newElement);

Number

All number are stored as an 64-bit floating bit number in JavaScript.

To conver a string a number, use the Number function. It return NaN if it failes and can be checked with the function isNaN(), example:

var foo = "55";
var myNumber = Number(foo);
if(isNaN(myNumber)) {
  console.log("It's not a number!")
}

There is a Math object that has many good methods for handle number like round(n), PI and cos(n);

Object

Objects are container for variables and function, these are called properties. An example below shows how to create an object:

var player = new Object();
player.name = "Fred";
player.score = 10000;
player.rank = 1;

or they can be created in a short hand like:

var player = { "name" : "Fred", "Score" : 10000, "rank" : 1};

Here is an example how to create a method to the object:

function playerDetails() {
  console.log(this.name + " has a rank of: " + this.rank)
}
player.logDetails = playerDetails;

Module

To call a function by it self it can be declared anonym like this example:

var MyModule = (function() {
  return {
    foo : function() {
      console.log('foo');
      return this;
    },
    bar = function() {
      console.log('bar');
      return this;
    }
  }
})();

This this often used in modules, the example above us a chaining so it can be called like:

MyModule.foo().bar().foo();

Event

Event that happen to children element, if it's not handled, then it'll be past to there parent's element.

Event listener can be added in two ways, by adding a function to the element's event or add it with the addEventListener(eventName, function, false);. Several event functions can be added to the same event using this method. An example:

element.onclick = function() { /* do somthing */ };
element.addEventListener('click', functionName, false);

Timer

There are two timers setTimeout(timeoutFunction, time); that will only be called once. There is also setInterval(intervalFunction, time); that will be called all the time with the given interval. The interval can then be stoped by using clearInterval(handle);.
