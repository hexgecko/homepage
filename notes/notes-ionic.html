AngularJS

To use the angular script add and ng-app to the head tag.

Controler To define a controller in AngularJS use the fallowing code:

function PlayerCtrl($scope) {
  $scope.players = {
    'Fred' = {
      'name' : 'Fred',
      'rank' : 2
    },
    'Mike' = {
      'name' : 'Mike',
      'rank' : 3
    }
  }
}

Where the $ says to the Angular that it's a controller.

The data and function in the scope can then be by adding the ng-controller to the parent element. An example how to interate the controller by adding an ng-controller="PlayerCtrl attribute in the parent (container) elemebt and do a ng-repeat="player in players" in a list control:

<div class="container" ng-controller="PlayerCtrl"gr;
  <ul>
    <li ng-repeat="player in players">{{player.name}} - {{player.rank}}</li>
  </ul>
</div>

Filter To add a filer to the result add an | in the placeholder with the filer name:

{{player.name | uppercase}}

There is a lot of other filter an they should be used for presentation when no changing the data.

The user can include a partial html file by adding the adding a ng-include src="scopePropertiesPath" to and div tag that span the where the html data should be displayed. But Ionic use the ui-router, so it's not applicable for Ionic.

Notes About Ionic

If you want to look and get an example there is a lot of examples here. The porject is create like this:

ionic start flickrApp http://codepen.io/ionic/pen/CrInw

Start a ionic server with the command:

ionic server -l

Where the -l flag display the apps for both IOS and Android devices in a browser.

Site for getting the name of the existing icons.

Ionic uses the MVC (Model View Control) Architecture where:

  • Model is the data used in the application
  • View is a template with html code
  • Control is JavaScript code written with AngularJS

Adding a ng-model="query" attribute to an input tag will put the query filled with the text out the input into the control $scope.

Ionic use ui-router to display different page within the application.

Add data to the link for sening information in the url like: _url: "/list/:aId"

A good thing to update the pages afterwards is to update the style sheet to update the styles.

Tools used by Ionic

  • AngularJS for the behaviour, (good to know how it works)
  • Cordova for distribution on IOS, Andriod and Microsoft Mobile
  • Sass for the look and it extends CSS, only have to know this if to change the theme.
  • Gulp
  • Git
  • Bower for installing things.

Ionic Widget

  • <ion-nav-bar> is used for the navigation bar
  • <ion-nav-view> is where the contains is displayed
