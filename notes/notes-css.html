What is CSS

CSS is the clothing for web pages.

Web Developer by Chris Pederick is a nice plugin for the web browser for getting information about the site.

The terminology about the css tag:

h1 { color: #333; font-size: 12px }

Where color and font-size is a property, #333 and 12px is values. Both the properties and value is a declaration and it should end with a ';'. The objects within the brackets are called a declaration block. The h1 is the selector, in this example, all h1 tag in the HTML document will be affected. All of the declaration is called a rule.

Selectors

If the selector starts with a '#', then it means it references to a tag with the id attributes. If the selector has the structure of h1#main_header the it's applicable to h1 that has the id of "main_header". To reference to a tag with a given class attribute a '.' is used. Example .first reference to all tag with the class attribute. To reference to a type of tag with the class attribute, the tag name comes before the '.', same as for the IDs. To change all tag of certain types within a tag, div for example.

#idname p { font-style: italic; }.

If two selectors apply to the same element, the one with the highest specificity wins. There are four distinct categories that define the level of a given selector: inline styles, IDs classes + attributes and elements. If the two have the same rules, the lastest win. Always try to use ids to increase the specificity and a class selector is always higher then an element selector.

* { } 0
li { } 1 (one element)
li:first-line { } 2 (one element, one pseudo-element)
ul li { } 2 (two elements)
ul ol+li { } 3 (three elements)
h1 + *[rel=up] { } 11 (one attribute, one element)
ul ol li.red { } 13 (one class, three elements)
li.red.level { } 21 (two classes, one element)
style="" 1000 (one inline styling)
p { } 1 (one HTML selector)
div p { } 2 (two HTML selectors)
.sith 10 (one class selector)
div p.sith { } 12 (two HTML selectors and a class selector)
#sith 100 (one id selector)
body #darkside .sith p { } 112 (HTML selector, id selector, class selector, HTML selector; 1+100+10+1)

To add an importance to a declaration will overwrite all another element with no important, an example:

h1 {background: green !important;}

This a good way to check is nothing changes depend on a specificity error or something else. However, don't forget to remove then afterward.

A style sheet can be impeded into the html document by adding a '<style type="text/css"/>' in the head tag. But the most common method is to link to file by adding a '<link rel="stylesheet" type="text/css" href="base.css"/ media="all">' tag within the head tag.

To import the other stylesheet into a stylesheet:

@import 'url("path-to-my-other.css);'

But they must come first within the style tag.

Selectors can be group together by putting a ',' between then in the declaration, an example:

h1, h2, h3 {text-transform: uppercase}

Box Model

Box model is the margin around the element, there are these properties: margin, border and padding. Here is an example:

p {margin: 1em; border: 1px solid #000; padding: 1em}

where an em, relative to the font size, so 1 is the same as the font size.

A good way to see the border of an element is change the color for the background.

Float

Simple floating: an image can float left or right.

This is an example how to use a content and a sidebar next to it:

#content {float: left; padding: 0 20em 4em 3 em}
#sidebar {float: right; width: 17em; margin: 0 0 4em -18em}

To but the footer at the bottom a clear properties can be used, here is an example:

#footer {clear: both}

To make the ordered list in the navigation to be next to each other without, one can use this trick:

#navlinks li {display: inline; list-style: none}

Colours

The ways to give colours with css is: #RRGGBB, rgb(128,200,432) or rgb(10%,20%,30%), an example for setting the for the body:

body {background-color: #E3EDC2}

A background image that is but side by side can be used to with the background-image, here is an example:

body {background-image: url(images/background.gif)}

Where the URL is a relative path. The side on side depends on the background-repeat that can have the fallowing values: repeat, repeat-x, repeat-y and no-repeat. The position of the image can be set with background-position. All of these can be but in just a background properties:

body {background: url(img/img.gif) repeat 0 0 rgb(85%,85%,0%)}

To change the padding between to the lines, a line-height can be used with the value is the different times the font size.

Font and Text

The font style and weight can be changed with the font-style and font-weight, in the example below give the font a tilt and make it bold.

p {font-style: italic; font-weight: bold}

To change the lower case letter to upper but smaller the upper case the fallowing declaration can be used:

h1 {font-variant: small-caps}

The font size can be changed in relative to its parent, like:

h1 {font-size: 2em}

The font can be changed by changing its family like this:

p {font-family: Verdana, Arial, Helvetica, sans-serif}
p {font-family: "Courier New", monospace}

The short form for setting the font is:

p {font: bold italic small-caps 2em/1.5em Arial, sans-serif}

Where the size and font style most come last, the 1.5em is the line-height

Alignment of the text can be used with the text-align: right|left|center|justify

Text can be transformed to upper or lower characters using the text-transform: uppercase|lowercase|capitalize

To make a line over, under or through can make the text-decoration with the values underline|overline|line-through

Here is an example how to create a navigation links with lines between then:

#navlinks {margin: 0; padding: 0.5em 3em}
#navlinks li {margin-right: 0.5em; border-right: 1px solid #99C; padding-right: 0.75em}
#navlinks li.last {border-right: 0}
