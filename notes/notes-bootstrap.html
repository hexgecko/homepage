Bootstrap

What is bootstrap?
  1. It's a framework.
  2. Mobile first.
  3. Responsive design.
  4. Battle tested - tested on many web browsers.

Can create a customised css/script "building" it on the bootstraps homepage.

Containers

  1. Fixed container, has breakpoints depended on the width.
  2. Fluid container, alwas fill the hole width.
  3. Row, place horizontal groups of columns and should be placed within an container.
  4. Column, placed in a row and have the format "col-SIZE-SPAN".

If the column format doesn't work by some reason, it can be reset with clearfix.

One can offset to put empty column for special sizes, col-xs-offset-3. Here the column will start at the 4th column.

Rows can be used within other rows to nest them.

Customise the order dependent on the screen width with push and pull, example col-xs-push-3.

Class attribute caption gives some extra padding.

Format Style and Alignment

Can style thing to look like an h1 by adding the class="h1" to your an element.

There are also some extra styles like: lead.

And some extra element: maked, smaller, s, del, ins, u, strong, b, i and em.

Alignment of text is user with the fpllowing classes text-left, textcenter_ and textright_ to align the text.

Changing the text formating can be user with the following classes textjustify, _textnowrap, _textunderline_ and textcapitalize_

Quote can be done with the blockquote element fallow by a p and footer element.

List

Unstyle list with the list-unstyled attribute.

Put item side by side can be done with the list-inline attribute.

Other list: list-group and panel

Button

Button should have the class attribute btn btn-[type] where the type can be: default, primary, success, info, warning and danger.

Change the size by giving a class attribute btn-[size] where size can be btn-lg, btn-sm and btn-xs for Large, Small and Extra Small.

Make the button blockable with the class btn-block attribute.

Change the state of the button with the class attribute btn-active and btn-disabled

Images

Make the image responsive with the class attribute img-responsive to center it use center-block attribute.

Rounded, thumbnail and circle edges with the class attribute: img-rounded, img-thumbnail and img-circle.

Media class attribute is a good way to format images.

Helper

Change the text format with the class attributes: text-muted, text-primary etc...

Change the backround with the class attributes: bg-primary etc...

Responsive Utilities

Have the format [visibility]-[size](-display) where:

  • Visibilty: visible or hidden
  • Size: xs, sm, md, lg or print
  • Display: block, inline, inline-block or print

Form

Use label with for attribute, importen for mobile and screen reader.

Format the input with the class attribute form-control and add them as a group with the other by putting them with in a div element with the class attribute form-group.

<div class="form-group">
  <label for="my-input">My Input:</label>
  <input type="text" id="my-input" placeholder="Your Name..."/>
</div>

Checkbox and Radio input would have the following format:

<div class="checkbox|radio">
  <label>
    <input type="checkbox|radio">
    My Checkbox|Radio
  </label>
</div>

Only have one Radio or Checkbox selected, add a name attribute with the same "name".

Can use column container within the form.

Bootstrap also have validation class attribute, read more on there homepage.

Navbar

Can have the navbar atop of everything with the attribute data-spy="affix" and the offset for when it will be display can be set with data-offset-top="100" attribute.
